# Radio Room Server

Express/Socket.io server for a [Radio Room](https://github.com/albatrocity/radio-room) instance.

```
# .env
CLIENT_ID=<SPOTIFY_CLIENT_ID>
CLIENT_SECRET=<SPOTIFY_CLIENT_SECRET>
REDIRECT_URI=http://localhost:3000/callback
```

## Development

```bash
docker compose up
```

Runs on port `3000`
